# Exercícios de CSS

Este repositório contém uma série de exercícios práticos para aprender e aplicar conceitos de **CSS**. Cada diretório corresponde a um exercício específico, abordando tópicos como layout, design responsivo e estilização de fundos.

## Estrutura do Repositório

```bash
/exercicios-css
├── 01-flexbox-layout
├── 02-responsive-grid
└── 03-css-background-props
```

## 01 – Flexbox Layout

Neste exercício, você aprenderá a utilizar o modelo de layout **Flexbox** para criar layouts flexíveis e responsivos. O Flexbox permite distribuir e alinhar elementos dentro de um container de forma eficiente.

### Objetivos
- Entender as propriedades básicas do Flexbox
- Criar um layout simples utilizando Flexbox
- Praticar alinhamento e distribuição de itens

## 02 – Responsive Grid

O objetivo deste exercício é aplicar o **CSS Grid** para criar layouts responsivos. O Grid Layout é uma técnica poderosa para controlar o posicionamento de elementos na página.

### Objetivos
- Compreender o funcionamento do CSS Grid
- Criar layouts com múltiplas colunas e linhas
- Adaptar o layout a diferentes tamanhos de tela

## 03 – CSS Background Properties

Neste exercício, você explorará as principais propriedades de fundo do CSS, como `background-color`, `background-image`, `background-size` e `background-position`.

### Objetivos
- Utilizar corretamente as propriedades de background
- Aplicar imagens de fundo e técnicas de posicionamento
- Explorar efeitos visuais e padrões de fundo

## Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/exercicios-css.git
   ```

2. Acesse o diretório do exercício desejado:
   ```bash
   cd exercicios-css/01-flexbox-layout
   ```
Abra o arquivo **index.html** no navegador para visualizar o exercício.
